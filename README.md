# YuZai Modpack 鱼崽的模组包

I got bored of vanilla REPO. So I configured the enemy spawn rates (such as the duck) to make it more interesting. Also included more shops items to balance the difficulty. Remember to disable the Chinese translation mod if you prefer english. Enjoy!

我玩腻了原版 REPO，所以调整了敌人刷新率（例如鸭子），让游戏更有趣。也加入了更多商店物品来平衡难度。玩得开心！

> *In some updates I remove mods due to bugs or repeated features. To ensure you have the same mods as your friends, it is suggested to redownload the entire modpack whenever there are major/minor version changes (e.g., from 1.2 → 1.3). You don’t need to redownload for small patch updates like 1.2.1 → 1.2.2.*
>
> *在某些更新中，我会删除部分模组因为bug或功能重复。为了确保你和朋友的模组一致，建议在主版本／次版本更新时（例如 1.2 → 1.3）重新下载整个模组包。对于 1.2.1 → 1.2.2 这种小补丁更新，则不需要重新下载。*

## Custom Configurations 自定义配置

To enable Vanilla spawn rates (normal spawn rates), disable the **[Spawn Config](https://thunderstore.io/c/repo/p/Index154/SpawnConfig/)** Mod. The enemy spawn rates increase as more players join. To balance this, **[More Shop Items](https://thunderstore.io/c/repo/p/Jettcodey/MoreShopItems_Updated/)** Mod have been configured to spawn more weapons. The truck has an item spawn limit so if you purchase too many items it will be queued in the "void". To retrieve these items, use the **[Trash Box](https://thunderstore.io/c/repo/p/XiaohaiMod/TrashBox/)** Mod to recycle unwanted items and the items in queue will appear next round. For more information on how to configure the enemy spawn rates, please check out [Spawn Config's Wiki](https://github.com/Index154/REPO_SpawnConfig/wiki).

如果你要玩原版的刷怪率（正常刷怪率），请关闭 **[刷怪配置](https://thunderstore.io/c/repo/p/Index154/SpawnConfig/)** 模组。随着玩家人数增加，刷怪率也会提高。为了平衡难度，**[更多商店物品](https://thunderstore.io/c/repo/p/Jettcodey/MoreShopItems_Updated/)** 模组已被设置刷出更多武器。卡车有物品数量上限，所以如果你购买太多物品，它们会被排在“虚空”中。若要取回这些物品，请使用 **[垃圾桶](https://thunderstore.io/c/repo/p/XiaohaiMod/TrashBox/)** 模组回收不需要的物品，队列中的物品会在下一轮出现。如需了解如何调整敌人刷怪率，请查看 [刷怪配置维基](https://github.com/Index154/REPO_SpawnConfig/wiki)。

## FAQ 常见问题

### The download is slow

This is due to the large assets from custom map mods. Downloading may take a while, depending on your internet speed and connection quality.

### The download keeps failing

This usually happens due to firewall (e.g., internet restrictions in China) or bad internet.

### Why only custom maps but no custom enemies/valuables/weapons?

You can't play with vanilla players. Custom maps can be controlled with MapVote so it's still possible to play with non-modded players.

### Why didn't you include Extraction Point Confirm Button?

Only players with this mod can see and use the extraction point button. So if you die, no one can extract the valuables.

### Can you include more custom maps?

Yes, I will add new maps whenever I get the chance to. The maps included have minimal to zero bugs. Some maps have carts falling through the floor into the abyss, while others have valuables spawning stuck in the map (Some are just bad maps sorry no offense). If you want a specific map which is not included here, just add it manually.

### 下载速度慢

这是因为自定义地图模组的资源文件较大。下载可能需要一些时间，具体多久取决于你的网速和网络稳定性。

### 下载总是失败

这通常是由于防火墙（例如中国的网络限制）或网络不稳定造成的。

### 为什么只有自定义地图，而没有自定义敌人 / 物品 / 武器？

因为那样会无法和原版玩家一起玩。自定义地图可以通过 **地图投票** 控制，所以仍然能够与未安装模组的玩家一起玩。

### 为什么没有加入“撤离点确认按钮”？

因为只有安装了这个模组的玩家才能看到并使用撤离点按钮。如果你死亡，其他人将无法带着贵重物品撤离。

### 你能加入更多自定义地图吗？

我会不断的加自定义地图。当前包含的地图都是没有bug的（或者不影响游戏）。有些地图会出现推车掉到地板下，坠入虚空的情况，也有些地图的贵重物品会卡在地图里（有些地图就是很敷衍，抱歉无冒犯）。如果你想要玩某个特定地图，请自行手动添加即可。
